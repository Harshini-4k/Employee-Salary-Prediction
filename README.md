\# Employee Salary Prediction System



\## Project Overview



This project is developed as part of the Machine Learning Internship Program.



The Employee Salary Prediction System predicts an employee's annual salary based on various attributes such as age, education level, work experience, job role, skills, certifications, company type, and weekly working hours.



The project demonstrates the complete machine learning workflow, including data generation, preprocessing, exploratory data analysis (EDA), visualization, model training, evaluation, and salary prediction.



\---



\## Features



\- Synthetic dataset containing 2000 employee records

\- Data preprocessing and validation

\- Exploratory Data Analysis (EDA)

\- Data visualization using Matplotlib and Seaborn

\- Machine Learning model using Linear Regression

\- Interactive salary prediction system

\- Model evaluation using MAE, RMSE, and R² Score



\---



\## Dataset Features



\- EmployeeID

\- Age

\- Gender

\- EducationLevel

\- YearsOfExperience

\- JobRole

\- SkillsScore

\- Certifications

\- CompanyType

\- WorkHoursPerWeek

\- Salary (Target Variable)
